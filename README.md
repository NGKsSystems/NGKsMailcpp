# NGKsMailcpp

Desktop mail client scaffold.

## Repository rules
- `third_party/` is quarantined for externally sourced code.
- `artifacts/` is runtime-only output and must stay gitignored.
- `docs/` stays concise and evolves by expanding existing docs.
